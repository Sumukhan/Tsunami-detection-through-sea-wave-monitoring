This is an ESP32-S2 based system that uses an accelerometer to monitor and detect sudden changes in sea waves. If there is such a change detected by accelerometer(s) then a graphical user interface displays a warning. The accelerometer sends it's recordings to a server, which also has a built-in GUI that displays the readings in real time, as well as the warnings. The accelerometer is programmed in Arduino IDE and the server is programmed in Python, using the Tkinter library. This system requires a network connection in order to send and receive data and both accelerometer and server have to be connected to the same network.
